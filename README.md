# Computer Evaluator
This is a mongodb project that I started mainly to practice design patterns. This app will take two computers that are listed or added to the database and compare them to tell the user which one is faster. 

#Patterns that have been implemented.
So far the patterns that have been implemented are the Model-View-Controller, DataMapper, and the Singleton pattern (for database access).
I may implement other patterns such as the factory or observer depending on how far I go with this.
